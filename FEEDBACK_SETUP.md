# Настройка отправки обратной связи

## Текущая реализация

Обратная связь отправляется через сервис FormSubmit (https://formsubmit.co/).

**Email получателя:** `vasil.ev81@mail.ru`

## Активация формы (один раз для каждого домена)

⚠️ **Важно:** Активация привязана к домену!

1. **На localhost (разработка):**
   - При первой отправке FormSubmit отправит письмо активации
   - Нажмите кнопку "ACTIVATE FORM" в письме
   - После этого форма будет работать на `localhost:5173`

2. **На GitHub Pages (продакшен):**
   - После деплоя на `tilkerman.github.io` нужно будет активировать форму заново
   - При первой отправке с продакшен-домена придет новое письмо активации
   - Нажмите "ACTIVATE FORM" в новом письме
   - После этого форма будет работать на GitHub Pages

3. **На телефоне (PWA):**
   - Если приложение установлено как PWA, оно использует тот же домен, что и веб-версия
   - Если домен уже активирован, форма будет работать автоматически

## Как это работает

1. Пользователь заполняет форму обратной связи
2. Данные отправляются на сервер FormSubmit
3. FormSubmit отправляет письмо на указанный email
4. После активации для домена все последующие письма приходят автоматически

## Переменные окружения (опционально)

- `VITE_FEEDBACK_EMAIL` - email получателя (по умолчанию: `vasil.ev81@mail.ru`)

## Тестирование

Для тестирования отправки:
1. Заполните форму обратной связи
2. Нажмите "Отправить"
3. Подтвердите отправку
4. При первой отправке с нового домена проверьте почту для активации
5. После активации все последующие письма будут приходить автоматически

## Альтернатива: EmailJS (если FormSubmit не работает)

Если FormSubmit не работает, можно использовать EmailJS:

1. Зарегистрируйтесь на https://www.emailjs.com/
2. Создайте Email Service (например, Gmail)
3. Создайте Email Template
4. Получите Public Key и Service ID
5. Добавьте в `.env`:
   ```
   VITE_EMAILJS_SERVICE_ID=your_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_template_id
   VITE_EMAILJS_PUBLIC_KEY=your_public_key
   ```
